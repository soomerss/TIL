# 연결리스트

## 추상적자료구조
DATA와 일련의 연산으로 이루어 졌고, 내부구조에 대해서는 알 필요가 없는 것

## Linked Lists 기본 구조
* 노드로 구성된다.
* 하나의 노드는 데이터 그리고 다음의 데이터가 무엇인지 담고 있는 Link로 이루어진다.
* 최초의 Head와 Tail 그리고 노드의 갯수를 정보로 가진다.

## 연산
* k번째 원소를 참조

## 배열과 비교

          배열         연결 리스트

특정원소 지칭 : 매우 간편   선형탐색과 유사
시간        : o(1)        o(n)
